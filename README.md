## Installation

1. Clone the repository:


2. Install the dependencies:

pip install -r requirements.txt


## Usage

1. Migrate the database:

python manage.py migrate

2. Run the development server:

python manage.py runserver


## Dependencies

- Django
- Pytz
- Datetime
